# Parcel action type codes
Each parcel action type has a specific code. \
A parcel is a set of changes that can be stacked onto a pallet. \
A parcel action is an interaction between an operator (i.e. user) and a parcel.

## Codes
You can find the codes of individual parcel action types in the table below.

| Code | Action type | Description                       |
|------|-------------|-----------------------------------|
| `1`  | `Author`    | Authored changes.                 |
| `2`  | `Stack`     | Stacked the parcel onto a pallet. |
